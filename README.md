# ИИ-ассистент для компании, оказывающей услуги по нострификации (признанию иностранных документов об образовании) для РФ

Ссылка на проект - https://t.me/nostrif_rf_bot

## Описание проекта

Проект представляет собой ИИ-ассистента, разработанного для автоматизации взаимодействия с клиентами компании, специализирующейся на услугах нострификации (признании иностранных документов об образовании) для Российской Федерации. Ассистент способен вести осмысленные диалоги с пользователями, отвечать на вопросы по базе знаний, записывать клиентов на консультации и интегрировать встречи в Google Календарь.

Основная цель проекта — улучшение качества обслуживания клиентов, повышение эффективности работы сотрудников и оптимизация процессов обработки запросов.

---

## Функционал

1. **Осмысленное ведение диалога**  
   Благодаря использованию современных языковых моделей (LLM), ассистент поддерживает естественные и логичные диалоги с клиентами. Это позволяет решать широкий спектр задач: от ответов на простые вопросы до помощи в сложных ситуациях.

2. **Ответы на вопросы по базе знаний**  
   Ассистент использует предварительно загруженную базу знаний, которая содержит информацию о процессе нострификации, требованиях к документам, стоимости услуг и других важных аспектах. Это гарантирует точность и актуальность предоставляемой информации.

3. **Запись на консультацию**  
   Клиенты могут легко записаться на консультацию через диалог с ассистентом. После подтверждения времени встречи информация автоматически добавляется в Google Календарь.

4. **Интеграция с Telegram**  
   Ассистент доступен в Telegram, что делает его удобным и доступным для пользователей. Клиенты могут задавать вопросы и получать помощь в привычном мессенджере.

## Используемые сервисы

1. **Qwen**  
   - Платформа для создания системного промпта и базы знаний.  
   - Позволила настроить логику работы ассистента и обучить его специфике бизнеса.

2. **Савви ([suvvy.ai](https://suvvy.ai/))**  
   - Сервис для создания мощных AI-ботов с возможностью работы по сложным сценариям.  
   - Обеспечивает высокую точность ответов (от 95%) и надежность работы на базе передовых языковых моделей (LLM).  
   - Поддерживает интеграцию с CRM, мессенджерами и другими платформами.

3. **Google Календарь**  
   - Используется для автоматической записи клиентов на консультации и управления расписанием.

4. **Telegram**  
   - Платформа для взаимодействия с клиентами.

## Шаги реализации

1. **Анализ требований и планирование**  
   - Изучение бизнес-процессов компании и выявление ключевых задач для автоматизации.  
   - Определение функционала ассистента: ответы на вопросы, запись на консультации, интеграция с Google Календарем.  

2. **Создание базы знаний**  
   - Сбор и структурирование информации о процессе нострификации, требованиях к документам, стоимости услуг и других аспектах.  
   - Настройка базы знаний в Qwen для обеспечения точности и актуальности ответов.  

3. **Разработка ассистента**  
   - Создание ИИ-бота с использованием платформы Савви.  
   - Настройка сценариев работы бота, включая обработку запросов, ведение диалогов и запись на консультации.  

4. **Интеграция с Google Календарем**  
   - Разработка API-интеграции для автоматического добавления встреч в Google Календарь.  
   - Тестирование функционала записи на консультации.  

5. **Запуск в Telegram**  
   - Создание Telegram-бота и подключение его к платформе Савви.  
   - Настройка интерфейса для удобного взаимодействия с клиентами.  

6. **Тестирование и доработка**  
   - Проведение тестирования ассистента на реальных сценариях использования.  
   - Внесение доработок на основе обратной связи от клиентов и сотрудников.  

7. **Запуск и мониторинг**  
   - Полноценный запуск ассистента в рабочем режиме.  
   - Мониторинг работы бота и анализ результатов для дальнейшего улучшения.

## Результаты проекта

- **Автоматизация коммуникаций:** Ассистент обрабатывает 100% обращений клиентов, предоставляя точные ответы в 95% случаев.
